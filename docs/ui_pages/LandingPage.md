## Overview

LandingPage is the public marketing page showcasing the platform's AI-first automation capabilities. It displays value propositions, platform capabilities, and CTAs for signup.

**Core Principles:**
- Public access (no auth required)
- Marketing-focused content
- AI-first messaging
- Responsive grid layouts
