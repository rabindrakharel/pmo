# ChatPage

**Version:** 9.0.0 | **Location:** `apps/web/src/pages/ChatPage.tsx` | **Updated:** 2025-12-03

---

## Overview

ChatPage provides an AI assistant chat interface for booking services, checking availability, and getting help. It displays info cards, example queries, and integrates the ChatWidget component.

**Core Principles:**
- AI assistant for service booking
- ChatWidget auto-opens on page load
- Example queries to guide users
- 24/7 availability indicator
